Adnroid Quiz-App made with Java and XML.
---
Contains 5 questions and each question contains 4 options displayed in Radio Buttons.
The questions, answers and options are stored in arrays, so minimal number of layouts is used.
After submission, the result displays as number of correct/incorrect attempts ans percentage.
